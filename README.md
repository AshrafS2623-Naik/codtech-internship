# CodTech Data Science Internship

This repository contains the tasks completed during the **CodTech Data Science Internship**.

## Tasks Completed

### Task 1: Data Pipeline Development

* Built a preprocessing pipeline using **Pandas** and **Scikit-learn**
* Handled missing values and feature scaling
* Trained a **Logistic Regression model** using the Titanic dataset

### Task 2: Deep Learning Project

* Implemented a **Neural Network** for image classification
* Used **PyTorch / TensorFlow**
* Dataset: MNIST handwritten digits

### Task 3: End-to-End Data Science Project

* Developed a **Flask API**
* Demonstrated how machine learning models can be deployed as web services

### Task 4: Optimization Model

* Solved a business optimization problem using **Linear Programming**
* Implemented using the **PuLP library**

## Technologies Used

* Python
* Pandas
* Scikit-learn
* PyTorch / TensorFlow
* Flask
* PuLP
* Matplotlib
* Seaborn
